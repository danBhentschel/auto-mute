When the toggle is set to "allow mode," only the pages you list will play sound while all other pages are muted. When it's set to "block mode," only the pages you list will be muted while all others play sound.

Specify a list of URLs where audio will be allowed. These pages will not be muted automatically and all pages not in the list will be muted automatically. Put one URL on each line. You can use the wildcard (*) character to indicate URL groups.
Specify a list of blocked URLs where audio will be blocked. These pages will be muted automatically and all pages not in the list will not be muted automatically. Put one URL on each line. You can use the wildcard (*) character to indicate URL groups.
NOTE: You can also use regular expression syntax to indicate URL groups if you surround the URL with backtick (`) characters.
Examples:
  • https://www.example.com/
  • https://www.example.com/*
  • https://*.example.com/*/page.html
  • `^https://.*\.?example\.com/.*$` ← Using regex
